"مدينتنا نظيفة" تربط بين **المواطن**، **الهيئات التنفيذية**، و**شركات التدوير** في منصة واحدة ذكية.

## ✨ الميزات

| الميزة | الوصف |
|--------|-------|
| 📸 **التبليغ الجغرافي (SIG)** | تصوير المفرغات وإرسال إحداثيات GPS فوراً |
| 📦 **الفرز الذكي** | طلب جمع الكرتون والبلاستيك مع نظام نقاط تحفيزي |
| 🚛 **تحسين المسارات** | توجيه الشاحنات للنقاط الأكثر احتياجاً |
| 🌱 **تتبع البصمة الكربونية** | حساب CO₂ الموفر من تحسين المسارات |
| 🤖 **ذكاء اصطناعي** | التنبؤ بمواعيد امتلاء الحاويات |
